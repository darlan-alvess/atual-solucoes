# Atual Soluções - Landing Page

Landing page desenvolvida para apresentação comercial da Atual Soluções, com foco em divulgar serviços de consultoria comercial, soluções em software de gestão e facilitar o contato direto pelo WhatsApp.

## Objetivo do projeto

O objetivo deste projeto foi criar uma página simples, responsiva e funcional para apresentar a atuação comercial da Atual Soluções, destacando seus serviços e facilitando o contato com potenciais clientes.

## Funcionalidades

- Layout responsivo para celular, tablet e desktop
- Menu de navegação por seções
- Seção inicial de apresentação
- Seção sobre a empresa
- Cards com soluções oferecidas
- Botões de contato pelo WhatsApp
- QR Code clicável para WhatsApp
- Link para o site oficial da empresa
- Efeitos visuais com hover
- Efeito de foco para acessibilidade
- Organização visual com variáveis CSS

## Tecnologias utilizadas

- HTML5
- CSS3
- Flexbox
- Media Queries
- Variáveis CSS

## Aprendizados

Durante o desenvolvimento deste projeto, pratiquei a criação de uma landing page utilizando HTML e CSS puro, aplicando conceitos como estrutura semântica, responsividade, organização de estilos, reutilização de classes, acessibilidade básica e boas práticas de manutenção com variáveis CSS.

## Status do projeto

Projeto concluído como versão inicial para portfólio.
